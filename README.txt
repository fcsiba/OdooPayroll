Follow these steps to use Odoo Payroll for Pakistan

1) Download Odoo from the following link and then install it.
   https://www.odoo.com/page/download
2) Now create a new database by clicking 'manage databases' and filling in the information
3) Copy the fine, penalties and HR extension module folders to the addons folder (C:\Program Files (x86)\Odoo 10.0\server\odoo\addons) 
4) Go to the settings and enable the 'activate the developer mode' option
5) Now click on apps and in the left pane select the 'update apps list' option
6) Search for the apps and install them
7) You will now be able to use these apps
